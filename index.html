Hanasaki Japan

Role
UI / UX Designer
Project
Travel & Digital Learning
Period
December 2021

01. Overview

Designed to provide a modern user experience, Hanasaki Japan is a travel blog with an empahsis on online Japanese learning and providing users with local recommendations. Users are able to send emails directly using an online email form and have the option to subscribe to a weekly newsletter.

02. Our Goal

My goal was to provide a design that seamlessly integrated a modern minimalist design with feature rich components. The prototype including all sections, pages and components was made in Figma and took about 3 days to complete.

03. User Story

This was designed with a mobile-first mindset as requested by the client. The mockup starts off at the initial landing page and is presented with a snippet of information for the rest of the website. The colour scheme of red and blue are used to create a modern, calming illustration feel. Red has been adopted for buttons and to draw attention to the user. Users should be able to transition smoothly between pages and select options without having to navigate to the hamburger menu.

The Coffee Project

Role
Front-End Developer & Engineer
Project
UI & UX Design, Website Development
Period
November,2021

01. Overview

Developed as a passion project, The Coffee Project is an advanced vanilla javascript web application that uses Geolocation, AJAX, Mapbox and Swiper.js to take the users location and give them hand curated coffee spot / cafe recommendations from their surroundings. Utilising local storage, the user then can save any coffee spot they wish into a favourites list so they can keep visiting their favourite again and again! Only available in Sydney, Australia.

02. Our Goal

My goal was to provide a seamless web application with modern language and coding architecture, as well as create a user experience that would be enjoyable and practical.

03. User Story

Inspired by the likes of Airbnb, Broadsheet and Google Maps, The Coffee Project has been developed to provide a seamless user experience.


Once on the homepage, the user is presented with homepage with a title and a "Get Started" button. This button will only work if location services are enabled on that persons device. Once done, they will be taken to a main page which is a presentation of the Mapbox API, where they will be presented with the different cafe locations on the map as well as a small title card at the bottom of the screen that gives them the option to "Get Directions" from their current position, "Add to List", or read more into the cafe, whereby a small description of the cafe will popup from the bottom.
